# Multi-Agent Research Assistant

AI-powered research tool using LangGraph multi-agent orchestration.

## Live Demo
https://multiagent1.streamlit.app

## Agent Architecture
- Search Agent — Web search via Tavily
- Summarizer Agent — Key findings extraction  
- Critic Agent — Fact verification with confidence scoring
- Writer Agent — Professional report generation

## Tech Stack
LangGraph · LangChain · Groq (Llama 3.3 70B) · Tavily · Streamlit
